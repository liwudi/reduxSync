# online-educationApp
基于react、react-router 4.0、antd-m、redux的App项目

## Start Project
>项目启动

第一步 

```
git clone https://github.com/liwudi/online-educationApp.git
```

第二步

```
npm install
```

第三步

```
npm start
```
